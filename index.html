隐私政策

MedCabinet - 保护您的数据安全

生效日期：2026年3月14日

🔒 隐私承诺

MedCabinet 非常重视用户的隐私保护。我们采用"隐私优先"的设计理念，您的药物数据完全由您掌控。我们不会在服务器上存储您的任何个人数据，也不会将您的数据出售给第三方。

我们收集的信息

MedCabinet 仅在您的设备本地存储以下信息：

  • 药物名称和描述
  • 药物有效期信息
  • 药物库存数量
  • 药物照片（由您选择添加）
  • 用药提醒设置

注意：以上所有数据默认仅存储在您的设备本地。

iCloud 同步

如果您选择开启 iCloud 同步功能：

  • 您的数据将通过 Apple 的 iCloud 服务进行同步
  • 数据在传输和存储过程中均经过加密
  • 我们不会在自有服务器上存储任何副本
  • 您可以随时在设置中关闭 iCloud 同步

数据使用方式

📱 本地存储

所有药物数据默认存储在您的设备本地，使用 iOS 系统的 Core Data 数据库技术。我们不收集、不上传、不分析您的任何数据。

🔔 本地通知

用药提醒和过期提醒使用 iOS 本地通知系统，无需网络连接，不会向任何服务器发送数据。

📷 照片访问

应用仅在您选择添加药物照片时访问相机或相册。照片存储在您的设备本地，不会上传至任何服务器。

📊 分析数据

我们不收集任何使用分析数据。您的使用习惯完全属于您的隐私。

第三方服务

MedCabinet 使用以下 Apple 原生服务：

  • iCloud：用于可选的数据同步功能
  • CloudKit：用于家庭成员间的数据共享（如开启）

这些服务受 Apple 隐私政策 的保护。

数据安全

我们采取以下措施保护您的数据安全：

  • 所有数据存储在设备本地，受 iOS 系统安全保护
  • 支持 iOS 设备加密和 Face ID/Touch ID 保护
  • iCloud 同步数据使用端到端加密
  • 不包含任何第三方追踪代码或广告 SDK

您的权利

作为用户，您拥有以下权利：

  • 访问权：随时查看应用中的所有数据
  • 导出权：将数据导出为 PDF 或 CSV 格式
  • 删除权：随时删除应用及其所有数据
  • 控制权：完全控制 iCloud 同步的开启和关闭

儿童隐私

MedCabinet 不面向 13 岁以下儿童。如果您是父母或监护人，发现您的孩子向我们提供了个人信息，请立即联系我们，我们将采取措施删除此类信息。

政策更新

我们可能会不时更新本隐私政策。任何更改将在应用内通知，并在本页面发布更新后的政策。继续使用我们的服务即表示您同意修订后的政策。
